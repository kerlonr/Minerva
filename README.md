# Minerva

Minerva é uma interface web simples para conversar com modelos de IA rodando localmente através do Ollama.

O projeto foi criado como um teste de chat com IA local, utilizando uma interface em HTML, CSS e JavaScript, com integração via API para comunicação com o Ollama.

## Funcionalidades

- Chat com modelos locais do Ollama
- Seleção de modelo disponível
- Renderização de respostas em Markdown
- Destaque de blocos de código
- Interface visual com animações
- Comunicação com o Ollama através da rota `/api/ollama`

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Ollama
- Marked.js
- Highlight.js
